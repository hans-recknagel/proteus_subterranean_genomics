# Proteus subterranean traits genomics

This repository accompanies:

> Recknagel et al. (in review)  
> *Phenotypic evolution of cave animals is consistently mirrored by extensive genomic convergence and remodelling*

The project investigates the genomics of subterranean adaptation and associated phenotypes across vertebrates, including:

- subterranean lifestyle
- pigmentation loss
- eye loss
- longevity

The study focuses primarily on the genome of the olm (*Proteus anguinus*) and comparative analyses across vertebrate taxa.

## Directory descriptions

### `scripts/`
Contains the R scripts used for statistical analyses, comparative genomic analyses, and figure generation.

### `data/`
Contains all input files and associated metadata required to reproduce the analyses.

### `results/`
Contains output files generated by the analyses.

# Running the analyses

Clone the repository:

```bash
git clone https://github.com/hans_recknagel/proteus_subterranean_genomics.git
```

Open the project in RStudio and run the scripts located in `scripts/`.

All file paths are relative to the repository root directory to ensure portability and reproducibility across systems.
